Documentation

Goal:

Eagleeye is a web-application with multiple search filters, an interactive map and a results section.

Home:

On the Home page, users can filter pg crime data by date, type, street number and street address. Users can either apply 1 of the filters, or two, or as many as they like. Instructions are straight forward. Apply a filter and click the GetData button. Markers will then populate the map with all the results showing the exact location where the crimes happened in PG county. Clicking a marker will display the street address.

About:

On the About page, users can learn about the project and the developer.
